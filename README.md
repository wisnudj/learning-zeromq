# learning-zeromq

This repository created for learning zeromq in c/c++.

book: ZeroMq

by: Faruk Akgul


### Installation

For zeromq installation, build source code ZeroMQ from git on Linux Ubuntu 18.04.

```sh
$ git clone https://github.com/zeromq/libzmq
$ ./autogen.sh
$ ./configure
$ make -j 4
$ make check
$ make install
$ sudo ldconfig
```
